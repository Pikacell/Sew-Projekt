Titel des Projekts: 2D-Kampfspiel – Klassenbasierter PvP-Modus

Projektbeschreibung:
Dieses Projekt ist ein lokal spielbares 2D-Kampfspiel in Java, entwickelt mit JavaFX. Es bietet einen 1v1-Modus, in dem zwei Spieler gegeneinander antreten können, nachdem sie aus zehn einzigartigen Charakterklassen ausgewählt haben. Das Spiel zeichnet sich durch eine einfache Steuerung, physikbasierte Projektile und abwechslungsreiche Spielmechaniken durch die verschiedenen Klassen aus.

Klassenwahl:
Vor dem Kampf wählt jeder Spieler eine von zehn spielbaren Klassen. Jede Klasse besitzt individuelle Eigenschaften wie Angriffsstil, Reichweite, Bewegungsgeschwindigkeit und zwei einzigartige Fähigkeiten.
Verfügbare Klassen:

- Swordsman
- Ninja
- Knight
- Mage
- Archer
- Royal Guard
- Spearman
- Berserker
- Monk
- Steuerung & Kampfmechanik:

Spieler 1:

Bewegung: W (vor), A (links), D (rechts)
Attacken: F (Attacke 1), G (Attacke 2)

Spieler 2:

Bewegung: Pfeiltasten ↑, ←, →
Attacken: K (Attacke 1), L (Attacke 2)

Das Ziel ist es, den Gegner mit gezielten Angriffen zu besiegen. Unterschiedliche Klassen ermöglichen verschiedene Spielstile wie Nahkampf, Fernkampf oder schnelle Bewegungen.

Physik und Projektilverhalten:
Projektile (z. B. Pfeile, Magiekugeln) unterliegen der Schwerkraft und verlieren mit der Zeit an Höhe. Es gibt eine physikbasierte Kollisionserkennung mit Spielern und Hindernissen, wobei diese mit JavaFX über die integrierten Animationen und Kollisionserkennungsmechanismen umgesetzt wird.

Technische Umsetzung:

Programmiersprache: Java
JavaFX (für die Darstellung und Spielmechanik)
Entwicklungsumgebung: IntelliJ IDEA
Grafiken: Eigene Pixel-Art
